Overview
++++++++++++++++++++

HSR-B 専用Gazeboプラグイン（共有ライブラリ）置き場。


プラグイン
+++++++++

* gazebo_hrh_gripper

   HSR-B 用グリッパを動かすためのプラグイン。

* hsrb_grasp_hack

   HSR-Bの吸引機構をシミュレータ上で実現するためのプラグイン。

* hsrb_bumper

   バンパセンサをシミュレータ上で実現するためのプラグイン。
